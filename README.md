# Worldly Treasures - Project #3

A responsive travel and e-commerce inspired layout built with pure HTML and CSS.

## Features
- Hero image header with navigation
- Blog post section with decorative styling
- Circular image gallery for travel destinations
- Product showcase section
- Contact form footer

## Notes
This is a static layout demo. Nav links, buttons, and the contact 
form are not functional — built to demonstrate responsive design 
and CSS layout skills only.
